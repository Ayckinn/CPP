# **SPYCIA v1.0 - For Linux**

### **HOW TO USE IT**
```./spysia``` or ```spysia```

To use ```spysia``` simple command,  you have to copy the file in the following directory :  
    
```sudo cp -v spysia /usr/local/bin```
