# **Moon Weather**

### **HOW TO USE IT**
```./MoonWeather```

*FOR RPi Raspberry OS*

1- Install Qt5-default :

```sudo apt install qt5-default```

2- Compile MoonWeather.pro

``` qmake MoonWeather.pro```

3- Make app

```make```
